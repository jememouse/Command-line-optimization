## 🎯 使用方法

### 快速开始
```bash
# 1. 给安装脚本执行权限
chmod +x install_dark_theme.sh

# 2. 运行安装脚本
./install_dark_theme.sh

# 3. 重启终端或重新加载配置
source ~/.zshrc
```

### 手动安装
```bash
# 直接加载配置文件
source enhanced_dark_theme.zsh
```
